# Cenipa-ReportAnalysis
This repository contains the code used to explore CENIPA's reports database

These files are for strictly academic use and must be used only for contributions to aviation safety. The raw data used for the construction of this repository were accessed on October 10, 2025 and are available at the following address:
https://sistema.cenipa.fab.mil.br/cenipa/paginas/relatorios/relatorios.php
